# Volleypool

Dit is een applicatie in ontwikkeling voor het vervoer voor een volleybalvereniging.

## Over Het Project
Het doel van dit project is om een alles-in-één oplossing te bieden voor de administratieve en organisatorische taken van het vervoeren van teams van de volleybalvereniging.

### Gebouwd Met
*   **C#**
*   **.NET Core**

## Functionaliteiten
* Registreren van een ouder
